# Quantum Computing Learning

This repositry contains my curated **Quantum Computing experiments and projects** using Qiskit.  
All code is written for **learning and exploration purposes**, with an emphasis on understanding fundamental concepts and building practical skills.

---

## Folder Structure

- **Learning-Exercises/** – Step-by-step practice exercises, small challenges, and solutions to strengthen quantum computing fundamentals.
- **Projects/** – More polished, structured projects demonstrating applied concepts.  

---

## Learning Objectives

- Build a strong foundation in qubits, gates, and quantum circuits.
- Gain hands-on experience with Qiskit simulation and experimentation.
- Explore quantum algorithms and their applications in classical and quantum machine learning.
- Demonstrate disciplined learning and problem-solving through structured exercises and projects.

This repository reflects a **learning journey**—from basic experiments to applied projects. While code is not production-level, it shows curiosity, growth, and a solid grasp of fundamentals.
